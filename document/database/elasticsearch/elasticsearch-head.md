# elasticsearch-head

一个es集群的web前端。

## 下载安装

- 安装之前需要node和git环境

- 配置淘宝镜像

- github地址：https://github.com/mobz/elasticsearch-head.git

- 下载依赖插件：`cnpm install`

- 修改es的配置，允许跨域。

  ``` properties
  http.cors.enabled: true
  http.cors.allow-origin: "*"
  ```

- 启动：`npm run start`
